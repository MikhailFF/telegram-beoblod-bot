# Telegram Morale Bot

Бот для Telegram-чата: если его тегнуть в сообщении, он отвечает короткой военной шуткой и ободряющим советом.

Шутки написаны оригинально, но по настроению опираются на флотскую, армейскую, медицинскую и авиационную прозу: Александра Покровского, Эдуарда Овечкина, Олега Дивова, Андрея Ситнянского, Валерия Меницкого, Андрея Ломачинского, Дмитрия Бочарова, Михаила Скрынникова, Александра Курышина, Владимира Бойко, Василия Ершова, Николая Черкашина, Владимира Каланова, Олега Химаныча, Алексея Суконкина, Михаила Драгунова, Ильи Деревянко, Алексея Моторова, Дениса Оканя, Виктора Конецкого, Сергея Колбасьева и Владимира Литтауэра.

Прямые цитаты и дословное содержание шуток из книг не используются. Бот хранит собственный банк коротких реплик по похожим темам: подплав, флотские казусы, штабная логика, армейский быт, ВДВ, спецназ, артиллерия, военная медицина и авиация.

## Установка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Установите зависимости:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e .
```

3. Скопируйте `.env.example` в `.env` и вставьте токен:

```powershell
Copy-Item .env.example .env
```

4. Запустите:

```powershell
telegram-morale-bot
```

Можно также запустить напрямую:

```powershell
python -m morale_bot.bot
```

## Проверка

```powershell
pip install -e ".[dev]"
python -m pytest
```

## Как пользоваться

Добавьте бота в групповой чат и напишите сообщение с его тегом, например:

```text
@your_bot_name подбодри нас
```

Также работают команды:

```text
/joke
/advice
/start
```

Для групп включите privacy mode у BotFather или оставьте его включенным: Telegram все равно доставляет боту команды и сообщения с упоминанием.
